AI Trading Project

طريقة التشغيل:

1) افتح المجلد في VS Code.
2) غيّر اسم الملف .env.example إلى .env.
3) ضع مفاتيحك داخل .env إذا أردت استخدام GPT أو TradersPost.
4) افتح Terminal واكتب:

npm.cmd install
npm.cmd start

رابط الاختبار:
http://localhost:3000

رابط Webhook من TradingView:
http://localhost:3000/webhook-tv

ملاحظة:
عند التشغيل المحلي لن يعمل TradingView Webhook مباشرة إلا إذا رفعت المشروع على Render/Railway أو استخدمت ngrok.
